A motor működése

Az HTML és a CSS elemek legenerálása is a kliens oldali JavaScrpitel történik.

A HTML elemek legenerálása

Az elem kiválasztására kattintva egy legördülő menüsorban ki tudjuk választani a HTML tag típusát. Ezt a JS kód másodpercenként 10-szer ellenőrzi, mivel a tag alapján jeleníti meg a tovább adatok bekérésére alkalmas input mezőket. Az ID és az input mezők kitöltése után tudjuk csak létrehozni, amit a háttérben futó kód egy random számkból álló un. keret DIV-el lát el amiben megtatlálható a "Törlés", "Keret", "Stílus beállítások" gombok. A keret gomb segítségével tudjuk eltüntetni az ezt a DIV-et jelző keretet, majd megjeleníteni.

A CSS beállítások legenerálása

A CSS beállításokat a "Stílus beállítások" gombra kattintva lehet állítani. Lehetőség van:

  • Betűszín beállítására
  • Háttérszín beállítására
  • Keret szín beállítására
  • A keret lekerekítésének beállítására
  • Belső és külső térköz beállítására
  • Betűméret beállítására

Ezeket a megadott ID alapját állítja be.

Gyors nézet

A szem ikonra kattintva lehet megjeleníteni a gyorsnézetet, ahol a kód ugyanúgy néz ki, mit egy böngészőben (csak kisebb). Ezt minden új HTMl elem hozzádásakor és minden új CSS beállításkor frissül.
