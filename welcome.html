Üdvözöllek!

Ezen a Weboldalon el tudod készíteni a saját kezdetleges statikus Weboldalad.

Alapvetően az oldal célja az, hogy egyszerűen és interakítvan tanítsa az érdeklődőkenek a HTML és CSS leíró nyelvet. Mivel az oldal használata közben látja, hogy milyen új HTML elemet és CSS beállítást hozott létre.

Az oldal használható egyszerű weblapok vagy blogok létrehezására.
